## Dataset Description
---

### health_fitness_dataset.csv
Raw CSV data from the Kaggle Health & Fitness dataset.

### fitness.rds
Processed version of `health_fitness_dataset.csv`.  
Includes:
- Imputed missing values for `health_conditions`
- Removal of invalid or inconsistent numerical entries

---

## Anomaly Detection Outputs

### Swimming Activity
- **kMeans_swimming.parquet**  
  Anomaly detection results for swimming data using the K-Means–based method.

- **oneClassSVM_swimming.parquet**  
  Anomaly detection results for swimming data using the One-Class SVM (OC-SVM) model.

- **autoencoder_swimming.parquet**  
  Anomaly detection results for swimming data generated by an autoencoder model.

---

### Running Activity
- **kMeans_running.parquet**  
  Anomaly detection results for running data using the K-Means–based method.

- **oneClassSVM_running.parquet**  
  Anomaly detection results for running data using the One-Class SVM (OC-SVM) model.

- **autoencoder_running.parquet**  
  Anomaly detection results for running data generated by an autoencoder model.
